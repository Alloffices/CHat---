Default interactions

Get Welcome Interaction

Returns Welcome Interaction object.

Request URL

Response example

{ “name”: “Welcome Interaction”, “responses”: [ { “type”: “text”, “filters”: [], “elements”: [ “Hi”, “Hello”, “What can I do for you?”, “Have a great day!” ] } ], “enabled”: true }

Properties

Property Type Description
name String Interaction name.
responses Array.<Object> An array of response objects.
enabled Boolean Indicates whether the welcome interaction is enabled or not.

Update Welcome Interaction

Updates the specified Welcome Interaction.

Request URL

Properties

Property Type Description
responses Array.<Object> An array of response objects.

Array(0, 10)
enabled Boolean Required. Indicates whether the welcome interaction is enabled or not.

Response example

{ “timestamp”: “2018-01-24T10:58:31.612Z”, “status”: { “code”: 200, “type”: “success” } }

Response example

Properties
